# @qualcomm-ui/css-utils

Utilities for bundling CSS in QUI projects.

This package provides CLI tools and utilities for processing, bundling, and optimizing CSS files with PostCSS, including support for nested CSS, minification, and autoprefixing.

For documentation and usage guides:

- [React documentation](https://react-next.qui.qualcomm.com/)
- [Angular documentation](https://angular-next.qui.qualcomm.com/)
