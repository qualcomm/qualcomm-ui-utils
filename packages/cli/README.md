# @qualcomm-ui/cli

Development utilities and scripts in a convenient CLI.

This package provides command-line tools for managing QUI packages, including publishing utilities, pre-publish hooks, and other development workflows.

For documentation and usage guides:

- [React documentation](https://react-next.qui.qualcomm.com/)
- [Angular documentation](https://angular-next.qui.qualcomm.com/)
